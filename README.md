# Calculadora Web

Aplicação web de calculadora desenvolvida com HTML, CSS e JavaScript, com interface moderna e suporte às operações matemáticas básicas.

## Acesso ao projeto
🔗 [Clique aqui para acessar](https://phmbezerra.github.io/calculadora-web/)

## Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão
- Limpar visor
- Apagar último caractere
- Exibir resultado da expressão digitada

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript

## Objetivo do projeto
Este projeto foi criado para praticar:
- construção de interfaces web
- manipulação de eventos
- lógica de operações matemáticas
- atualização dinâmica de elementos na tela
- organização de layout responsivo

## Como executar
1. Baixe ou clone este repositório
2. Abra a pasta do projeto
3. Execute o arquivo `index.html` no navegador

index.html

README.md

## Aprendizados
Durante o desenvolvimento deste projeto, foram praticados conceitos como:
- captura de cliques em botões
- construção e exibição de expressões
- tratamento de erros
- atualização do visor em tempo real
- separação entre estrutura, estilo e comportamento
